# TOTV PLUS — موقع التطبيق الترفيهي

موقع تعريفي وتسويقي لتطبيق **TOTV PLUS** (أفلام / مسلسلات / بث مباشر) بهوية بصرية سوداء وذهبية.
يسحب البوسترات والأعمال والممثلين وسيرهم الذاتية وقصص الأعمال تلقائياً من قاعدة بيانات الأفلام (TMDB)،
ويعرض أزرار تحميل لكل جهاز (أندرويد / آيفون / الشاشات الذكية).

الموقع **ثابت بالكامل (HTML/CSS/JS)** ولا يحتاج أي خادم أو خطوة بناء — جاهز للنشر مباشرة على GitHub Pages.

```
totvplus-site/
├── index.html              ← الصفحة الرئيسية
├── .nojekyll               ← يمنع GitHub من معالجة الملفات
├── README.md
└── assets/
    ├── css/style.css       ← التصميم (أسود/ذهبي + متجاوب)
    └── js/
        ├── config.js       ← المفتاح وروابط التحميل (عدّل هنا فقط)
        └── app.js          ← منطق جلب الأفلام والممثلين
```

---

## النشر على GitHub Pages (خطوة بخطوة)

1. أنشئ مستودعاً (Repository) جديداً على GitHub، مثلاً باسم `totvplus`.
2. ارفع **محتويات** مجلد `totvplus-site` إلى جذر المستودع (وليس المجلد نفسه).
   - عبر الموقع: زر **Add file → Upload files** ثم اسحب الملفات.
   - أو عبر الأوامر:
     ```bash
     git init
     git add .
     git commit -m "TOTV PLUS site"
     git branch -M main
     git remote add origin https://github.com/USERNAME/totvplus.git
     git push -u origin main
     ```
3. من المستودع: **Settings → Pages**.
4. في **Source** اختر الفرع `main` والمجلد `/ (root)` ثم **Save**.
5. انتظر دقيقة، وسيظهر رابط الموقع بالشكل:
   `https://USERNAME.github.io/totvplus/`

## التعديل على الإعدادات

كل ما قد تحتاج تغييره موجود في ملف واحد: `assets/js/config.js`

| الإعداد | الوصف |
|---|---|
| `API_KEY` | مفتاح قاعدة بيانات الأفلام (TMDB). |
| `DOWNLOAD.DOWNLOADER_CODE` | رمز تطبيق Downloader لأندرويد والشاشات الذكية (الحالي: `2606779`). |
| `DOWNLOAD.IOS` | رابط متجر آبل لتطبيق الآيفون. |

## ملاحظات مهمة

- **المفتاح ظاهر في الكود**: هذا طبيعي لأي موقع ثابت يستخدم TMDB من جهة المتصفح. مفتاح TMDB مخصّص للقراءة فقط.
  لو رغبت في إخفائه تماماً ستحتاج وسيطاً (Proxy) على خادم، وهو خارج نطاق الموقع الثابت.
- **اللغة**: يحاول الموقع جلب العناوين والقصص والسير بالعربية، وعند عدم توفّر الترجمة يعرض النسخة الإنجليزية تلقائياً.
- **متجاوب بالكامل**: يعمل على الجوال وأندرويد والكمبيوتر، مع تكبير خاص لشاشات التلفاز الكبيرة ودعم التنقّل بالريموت.
- **البث المباشر**: قسم "يُعرض الآن" يعرض الأعمال على الهواء من قاعدة الأفلام. لربط قنوات بث فعلية يلزم مصدر بثّ خاص بك.
