# توتيفي TOTV — موقع totv.app (النسخة الحديثة)

موقع توتيفي معاد تصميمه بهوية بصرية سوداء/ذهبية حديثة، مع الحفاظ على نفس تنظيم الموقع القديم:
الأقسام (أفلام، مسلسلات، الرياضة، النجوم، الباقات، التحميل، التواصل)، الدومين، الأيقونات،
تحليلات Google، بيانات SEO، الباقات بالدينار، رابط الدفع، والواتساب — كلها محفوظة.

الموقع **ثابت بالكامل** وجاهز للنشر مباشرة على GitHub Pages (بدون أي خطوة بناء).

## البنية
```
.
├── index.html              ← الصفحة الرئيسية (كل الأقسام + زر تحميل لكل الأجهزة)
├── download.html           ← صفحة التحميل التفصيلية + طريقة التثبيت + الأسئلة
├── CNAME                   ← الدومين totv.app
├── app-ads.txt             ← إعلانات Google (كما هو)
├── sitemap.xml · robots.txt
├── .nojekyll
├── logo.png · logo44.png · Icon-16/32/180/192/512.png
└── assets/
    ├── css/style.css       ← التصميم الحديث (أسود/ذهبي + متجاوب)
    └── js/
        ├── config.js       ← المفتاح + روابط التحميل + الباقات + التواصل (عدّل هنا فقط)
        └── app.js          ← جلب الأفلام/المسلسلات/النجوم/القصص + بناء الباقات
```

## التحميل لكل جهاز (مضبوط في `assets/js/config.js`)
- **أندرويد**: ملف APK الرسمي من GitHub Releases.
- **آيفون/آيباد**: متجر آبل `id6443335504`.
- **الشاشات الذكية**: تطبيق Downloader برمز `2606779`.
- **المباريات على iOS**: Smarters Player Lite (كما في الموقع القديم).
- زر «حمّل التطبيق لكل الأجهزة» ظاهر في الواجهة الرئيسية وينقل لصفحة التحميل.

## النشر على GitHub Pages
1. ارفع **محتويات** هذا المجلد إلى جذر المستودع.
2. **Settings → Pages** → الفرع `main` والمجلد `/ (root)` → Save.
3. لأن ملف `CNAME` موجود، سيرتبط الموقع بدومين **totv.app** مباشرة (تأكّد من إعدادات DNS لديك).
